Labosi iz MTTPP za referencu za projektni zadatak

LV4 
Lab 4 is covering automatic testing of Android (mobile) application using Android Emulator (Pixel 6, API 31), Java Development Kit (JDK), Apache Maven (manage project ie. build, resolve dependencies, ensure project layout and add plugins), Selenium to execute test actions and Appium (open source testing framework for mobile apps Android/iOS). Project is relying on Node.js to run the server. Project is created in IntelliJ.

Under Issues (in this repo) is a video record of the screen showing appium server log, mvn test log and Android emulator.
